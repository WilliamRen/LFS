脚本语言使用 lua，why?因为快且小，而且非常简单<br>

#####设置端口号
LFS --port[-p] 9002<br>

#####设置文件存储目录
LFS --dir[-d] Storage<br>

#####打开打印模式
LFS --print<br>

#####守护进程
LFS --daemon<br>

#####设置 Socket 的收发缓存大小
LFS --inBufferSize 102400 --outBufferSize 102400<br>
